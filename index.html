With the right guidance I've chosen SuperpoweredSDK to develop my personal project. I am currently employed as a Cyber Engineer and am working on CompTIA Network+ & Azure certificates.

I call my DAW project the "audioBuilder", composed of three parts.

  1. The processor, coded in c++. Use CPU to modify audio file (.mp3)
  2. The Controller, coded in JS. Running on Node env, the controller makes calls to the processor and store files in /tmp/
  3. The View, coded in JS with React / Redux and something called Fusion.

As you can see, it's pretty much MVC except the model is audio. All parts communicate with eachother using JSON. Repositories posted to github.

This project has been almost two years now in the making! I'm currently fixing time delay effects on the audioBuild exec whilst getting a grasp of front-end development. These skills will be put to use for pen-testing, so I can deploy my own servers / websites and use Kali Linux to test for vulnerabilities.
